# Voice Dictation

Локальный голосовой ввод с AI-коррекцией для Windows. Говоришь в микрофон — текст появляется в любом приложении.

Работает полностью на твоём компьютере, ничего не отправляет в интернет. Аналог [SuperWhisper](https://superwhisper.com), но бесплатный и open-source.

**Как это работает:** Нажал горячую клавишу → надиктовал → нажал ещё раз → программа распознаёт речь → AI исправляет ошибки и пунктуацию → готовый текст вставляется туда, где стоит курсор.

## Возможности

- **Два движка распознавания речи на выбор:**
  - **Whisper** — быстрый старт (~3 сек), умеет сам определять язык, поддерживает 99+ языков
  - **NVIDIA Canary** — потребляет меньше видеопамяти (~6 GB вместо ~10 GB), 25 европейских языков
- **AI-коррекция текста** — после распознавания нейросеть исправляет ошибки, расставляет знаки препинания и заглавные буквы, IT-термины пишет на английском (Docker, API, git и т.д.)
- **Коррекцию можно отключить** — если хочется получать "сырой" текст без обработки
- **Удобный интерфейс** — маленький виджет поверх всех окон показывает текущий статус (запись / распознавание / коррекция / готово)
- **Настройки через интерфейс** — правый клик по виджету: выбор движка, модели, языка, горячих клавиш
- **История диктовок** — все распознанные тексты сохраняются; клик по записи — копирует в буфер обмена
- **Полностью локально** — никакие данные не уходят в облако

## Что нужно для работы

| Компонент | Требования |
|-----------|------------|
| **Видеокарта** | NVIDIA с 10+ GB видеопамяти (например, RTX 3080, 4070 и выше) |
| **CUDA** | Версия 12 или новее (устанавливается вместе с драйверами NVIDIA) |
| **Оперативная память** | 16 GB и больше |
| **Система** | Windows 10 или 11 |
| **Python** | 3.12 |

### Сколько видеопамяти занимает

| Конфигурация | Распознавание | Коррекция | Итого |
|-------------|---------------|-----------|-------|
| Whisper + Qwen3.5 9B | ~3 GB | ~6.6 GB | **~10 GB** |
| Whisper + Qwen3.5 4B | ~3 GB | ~3.4 GB | **~6.4 GB** |
| Canary + Qwen3.5 9B | ~6 GB | ~6.6 GB | **~12.7 GB** |

> Если видеопамяти не хватает — выбери модель коррекции поменьше (например, `qwen3.5:4b` вместо `9b`).

## Установка

### Шаг 1. Python и виртуальное окружение

Открой **PowerShell** и выполни:

```powershell
pip install uv
uv python install 3.12
uv venv whisper-env --python 3.12
~\whisper-env\Scripts\Activate.ps1
```

> `uv` — это быстрый менеджер пакетов Python. Виртуальное окружение нужно, чтобы не засорять систему.

### Шаг 2. PyTorch с поддержкой видеокарты

```powershell
uv pip install --reinstall torch torchaudio --index-url https://download.pytorch.org/whl/cu128
```

> Эта команда устанавливает PyTorch с поддержкой CUDA (вычисления на видеокарте). Без этого распознавание будет работать только на процессоре и будет очень медленным.

### Шаг 3. Зависимости программы

```powershell
# Обязательные
uv pip install numpy miniaudio keyboard pyperclip httpx

# Движок распознавания — Whisper (рекомендуется)
uv pip install faster-whisper

# Движок распознавания — Canary (опционально, если хочешь попробовать)
uv pip install nemo_toolkit[asr] silero-vad soundfile
```

### Шаг 4. Ollama — локальный AI для коррекции текста

[Ollama](https://ollama.com) — это программа, которая запускает языковые модели (нейросети) локально на твоём компьютере. Она нужна для коррекции распознанного текста.

```powershell
# Установить Ollama
winget install Ollama.Ollama

# Скачать модель для коррекции (рекомендуется)
ollama pull qwen3.5:9b
```

> После установки Ollama работает в фоне (иконка в трее). Она должна быть запущена, когда ты используешь диктовку.

### Шаг 5. Клонировать репозиторий

```powershell
git clone https://github.com/arigandores/voice-dictation.git
```

### Шаг 6. Проверка

```powershell
~\whisper-env\Scripts\Activate.ps1

# Видеокарта доступна?
python -c "import torch; print(f'CUDA: {torch.cuda.is_available()}, GPU: {torch.cuda.get_device_name(0)}')"

# Whisper установлен?
python -c "from faster_whisper import WhisperModel; print('Whisper OK')"

# Ollama работает?
curl http://localhost:11434/api/tags
```

Если все три команды выполнились без ошибок — всё готово.

## Запуск

```powershell
~\whisper-env\Scripts\Activate.ps1
cd voice-dictation
python main.py
```

При запуске появится маленький виджет в правом верхнем углу экрана. Модель загрузится за несколько секунд, после чего виджет покажет "Готов к записи".

## Как пользоваться

| Действие | Что делать |
|----------|-----------|
| **Начать запись** | Нажать горячую клавишу (по умолчанию `Ctrl+Shift+Space`) |
| **Остановить запись** | Нажать ту же клавишу ещё раз |
| **Открыть настройки** | Правый клик по виджету → Настройки |
| **Посмотреть историю** | Правый клик по виджету → История |
| **Выйти** | `Ctrl+Shift+Q` или правый клик → Выход |

### Пошагово

1. Открой любое приложение (браузер, мессенджер, редактор, документ)
2. Поставь курсор туда, куда нужно вставить текст
3. Нажми `Ctrl+Shift+Space` — виджет станет красным, запись пошла
4. Говори в микрофон
5. Нажми `Ctrl+Shift+Space` ещё раз — запись остановится
6. Подожди ~3 секунды — текст автоматически вставится в то место, где стоит курсор

## Настройки

Все настройки доступны через виджет: **правый клик → Настройки**.

| Параметр | Что это | По умолчанию |
|----------|---------|-------------|
| **Движок** | Какой движок распознаёт речь: Whisper (быстрый, универсальный) или Canary (экономит видеопамять) | Whisper |
| **Модель Whisper** | Какая именно модель Whisper используется. `large-v3` — стандартная, скачивается автоматически | `large-v3` |
| **Коррекция через LLM** | Включить/выключить AI-коррекцию текста после распознавания | Включена |
| **Модель коррекции** | Какая нейросеть исправляет текст. Список подтягивается из Ollama автоматически | `qwen3.5:9b` |
| **Язык** | Язык распознавания. "Авто" — программа сама определит язык | Авто |
| **Горячие клавиши** | Какие клавиши нажимать для записи и выхода | `Ctrl+Shift+Space` / `Ctrl+Shift+Q` |

> При сохранении настроек модели перезагружаются автоматически — перезапускать программу не нужно.

## Альтернативные модели

### Модели распознавания (Whisper)

| Модель | Размер | Скорость | Примечание |
|--------|--------|----------|------------|
| `large-v3` | ~3 GB | Средняя | Стандартная, скачивается автоматически |
| `large-v3-turbo` | ~1.6 GB | Быстрая | Легче и быстрее, качество почти такое же |

Чтобы сменить — зайди в настройки и выбери из списка.

### Модели коррекции (Ollama)

| Модель | Время ответа | Видеопамять | Качество |
|--------|-------------|-------------|----------|
| `gemma3:4b` | ~2.3 сек | 3.3 GB | Хорошее |
| `qwen3.5:4b` | ~2.4 сек | 3.4 GB | Хорошее |
| **`qwen3.5:9b`** | **~2.5 сек** | **6.6 GB** | **Лучшее** |

Чтобы добавить модель:
```powershell
ollama pull gemma3:4b
```
После этого она появится в списке в настройках.

## Решение проблем

### "Ollama не отвечает"

Ollama должна работать в фоне. Проверь:
```powershell
curl http://localhost:11434/api/tags
```
Если не отвечает — найди Ollama в меню "Пуск" и запусти.

### "CUDA не найдена" или распознавание очень медленное

PyTorch установлен без поддержки видеокарты. Переустанови:
```powershell
uv pip install --reinstall torch torchaudio --index-url https://download.pytorch.org/whl/cu128
```

### Предупреждения про Triton / Megatron / OneLogger

Это нормально для Windows. На работу программы не влияет.

### Текст вставляется не туда

Текст вставляется туда, где сейчас стоит курсор (через `Ctrl+V`). Убедись, что курсор в нужном поле **до** начала записи.

### Микрофон не работает

Проверь настройки звука Windows — убедись, что нужный микрофон установлен устройством по умолчанию, или выбери его в настройках программы (правый клик → Настройки → Микрофон).

## Как это работает (для разработчиков)

```
Микрофон → [Silero VAD*] → Whisper/Canary → [Ollama LLM*] → Ctrl+V
  16kHz       *только         распознавание    *опционально     вставка
  mono        Canary          речи на GPU      коррекция        в активное
                                               текста           окно
```

1. **Запись** — `miniaudio` захватывает аудио с микрофона (16 кГц, моно)
2. **Удаление тишины** (только Canary) — Silero VAD вырезает паузы
3. **Распознавание** — Whisper или Canary обрабатывает аудио на GPU
4. **Коррекция** (если включена) — LLM исправляет ошибки и расставляет пунктуацию
5. **Вставка** — текст копируется в буфер обмена и вставляется через эмуляцию `Ctrl+V`

### Структура проекта

```
main.py                  # Точка входа
dictation/
    config.py            # Константы и настройки
    state.py             # Общее состояние (конфиг, локи, история)
    vad.py               # Silero VAD (фильтрация тишины)
    asr.py               # Движки распознавания (Whisper, Canary)
    llm.py               # Коррекция через Ollama
    audio.py             # Запись и пайплайн обработки
    hotkeys.py           # Горячие клавиши
    ui/
        overlay.py       # Виджет статуса
        history.py       # Окно истории
        settings.py      # Окно настроек
```
